Para usar sqlite3 no visual studio é necessario:

-Incluir o sqlite3.c às sources;

-Ir à aplicação -> Manage Package NuGet -> install sqlite3_c_plus_plus do Dawlin Peña L. & Allam Gallan

- Nota se fizerem isto, só precisam de incluir no vosso projeto o sqlite3.c e podem apagar a pasta packages que vai resultar da operação anterior.
